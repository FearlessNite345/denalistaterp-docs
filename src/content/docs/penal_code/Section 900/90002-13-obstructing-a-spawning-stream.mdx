# 900.02-13 - OBSTRUCTING A SPAWNING STREAM

This offense is defined as: Obstructing A Spawning Stream.

<Aside type="caution" title="Charge Type: Misdemeanor">
- Classified as a **Misdemeanor** under this code entry.
- Driver license points on conviction: **0**.
- Base fine: **$1,500** (before fees/taxes if any).
</Aside>

## Elements
- Operates or controls a motor vehicle or engages in the described conduct.
- Performs the behavior described by the violation title.

## Penalties
- **Fine**: $1,500
- **License Points**: 0
