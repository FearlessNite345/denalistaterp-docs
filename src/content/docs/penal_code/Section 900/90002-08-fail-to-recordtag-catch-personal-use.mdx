# 900.02-08 - FAIL TO RECORD/TAG CATCH (PERSONAL USE)

This offense is defined as: Fail To Record/Tag Catch (Personal Use).

<Aside type="tip" title="Charge Type: Infraction">
- Classified as a **Infraction** under this code entry.
- Driver license points on conviction: **0**.
- Base fine: **$400** (before fees/taxes if any).
</Aside>

## Elements
- Operates or controls a motor vehicle or engages in the described conduct.
- Performs the behavior described by the violation title.

## Penalties
- **Fine**: $400
- **License Points**: 0
