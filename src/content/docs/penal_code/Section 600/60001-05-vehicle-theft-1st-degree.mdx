# 600.01-05 - VEHICLE THEFT 1st Degree

This offense is defined as: Vehicle Theft 1St Degree.

<Aside type="danger" title="Charge Type: Felony">
- Classified as a **Felony** under this code entry.
- Driver license points on conviction: **0**.
- Base fine: **$5,000** (before fees/taxes if any).
</Aside>

## Elements
- Operates or controls a motor vehicle or engages in the described conduct.
- Performs the behavior described by the violation title.

## Penalties
- **Fine**: $5,000
- **License Points**: 0
