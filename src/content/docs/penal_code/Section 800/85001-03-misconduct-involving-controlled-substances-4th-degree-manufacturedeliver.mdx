# 850.01-03 - MISCONDUCT INVOLVING CONTROLLED SUBSTANCES 4th Degree (MANUFACTURE/DELIVER)

This offense is defined as: Misconduct Involving Controlled Substances 4Th Degree (Manufacture/Deliver).

<Aside type="danger" title="Charge Type: Felony">
- Classified as a **Felony** under this code entry.
- Driver license points on conviction: **0**.
- Base fine: **$3,500** (before fees/taxes if any).
</Aside>

## Elements
- Operates or controls a motor vehicle or engages in the described conduct.
- Performs the behavior described by the violation title.

## Penalties
- **Fine**: $3,500
- **License Points**: 0
