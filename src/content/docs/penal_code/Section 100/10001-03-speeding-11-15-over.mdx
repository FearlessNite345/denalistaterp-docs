# 100.01-03 - SPEEDING 11-15 OVER

Operating a motor vehicle above the posted speed limit by 11 to 15 mph.

<Aside type="tip" title="Charge Type: Infraction">
- Classified as a **Infraction** under this code entry.
- Driver license points on conviction: **2**.
- Base fine: **$225** (before fees/taxes if any).
</Aside>

## Elements
- Operates or controls a motor vehicle or engages in the described conduct.
- On a public roadway subject to a posted speed limit.
- Exceeds the posted speed limit by between 11 and 15 mph, inclusive.

## Penalties
- **Fine**: $225
- **License Points**: 2

## Example
Example: Driving 13 mph over the limit in a posted zone.
