# 100.01-04 - SPEEDING 16-20 OVER

Operating a motor vehicle above the posted speed limit by 16 to 20 mph.

<Aside type="caution" title="Charge Type: Misdemeanor">
- Classified as a **Misdemeanor** under this code entry.
- Driver license points on conviction: **3**.
- Base fine: **$300** (before fees/taxes if any).
</Aside>

## Elements
- Operates or controls a motor vehicle or engages in the described conduct.
- On a public roadway subject to a posted speed limit.
- Exceeds the posted speed limit by between 16 and 20 mph, inclusive.

## Penalties
- **Fine**: $300
- **License Points**: 3

## Example
Example: Driving 18 mph over the limit in a posted zone.
