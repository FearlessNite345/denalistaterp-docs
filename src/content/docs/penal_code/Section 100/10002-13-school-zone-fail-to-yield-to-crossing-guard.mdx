# 100.02-13 - SCHOOL ZONE: FAIL TO YIELD TO CROSSING GUARD

This offense is defined as: School Zone: Fail To Yield To Crossing Guard.

<Aside type="caution" title="Charge Type: Misdemeanor">
- Classified as a **Misdemeanor** under this code entry.
- Driver license points on conviction: **3**.
- Base fine: **$350** (before fees/taxes if any).
</Aside>

## Elements
- Operates or controls a motor vehicle or engages in the described conduct.
- Performs the behavior described by the violation title.

## Penalties
- **Fine**: $350
- **License Points**: 3
