# 100.01-01 - SPEEDING 1-5 OVER

Operating a motor vehicle above the posted speed limit by 1 to 5 mph.

<Aside type="tip" title="Charge Type: Infraction">
- Classified as a **Infraction** under this code entry.
- Driver license points on conviction: **0**.
- Base fine: **$75** (before fees/taxes if any).
</Aside>

## Elements
- Operates or controls a motor vehicle or engages in the described conduct.
- On a public roadway subject to a posted speed limit.
- Exceeds the posted speed limit by between 1 and 5 mph, inclusive.

## Penalties
- **Fine**: $75
- **License Points**: 0

## Example
Example: Driving 3 mph over the limit in a posted zone.
