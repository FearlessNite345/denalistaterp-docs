# 100.02-14 - PASSING A SCHOOL BUS (RED LIGHTS ACTIVATED)

This offense is defined as: Passing A School Bus (Red Lights Activated).

<Aside type="caution" title="Charge Type: Misdemeanor">
- Classified as a **Misdemeanor** under this code entry.
- Driver license points on conviction: **5**.
- Base fine: **$650** (before fees/taxes if any).
</Aside>

## Elements
- Operates or controls a motor vehicle or engages in the described conduct.
- Performs the behavior described by the violation title.

## Penalties
- **Fine**: $650
- **License Points**: 5
