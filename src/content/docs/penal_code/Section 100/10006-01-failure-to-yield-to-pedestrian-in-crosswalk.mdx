# 100.06-01 - FAILURE TO YIELD TO PEDESTRIAN IN CROSSWALK

This offense is defined as: Failure To Yield To Pedestrian In Crosswalk.

<Aside type="tip" title="Charge Type: Infraction">
- Classified as a **Infraction** under this code entry.
- Driver license points on conviction: **3**.
- Base fine: **$300** (before fees/taxes if any).
</Aside>

## Elements
- Operates or controls a motor vehicle or engages in the described conduct.
- Performs the behavior described by the violation title.

## Penalties
- **Fine**: $300
- **License Points**: 3
