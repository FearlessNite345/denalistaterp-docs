# 100.01-06 - SPEEDING 26-34 OVER

Operating a motor vehicle above the posted speed limit by 26 to 34 mph.

<Aside type="caution" title="Charge Type: Misdemeanor">
- Classified as a **Misdemeanor** under this code entry.
- Driver license points on conviction: **4**.
- Base fine: **$450** (before fees/taxes if any).
</Aside>

## Elements
- Operates or controls a motor vehicle or engages in the described conduct.
- On a public roadway subject to a posted speed limit.
- Exceeds the posted speed limit by between 26 and 34 mph, inclusive.

## Penalties
- **Fine**: $450
- **License Points**: 4

## Example
Example: Driving 30 mph over the limit in a posted zone.
