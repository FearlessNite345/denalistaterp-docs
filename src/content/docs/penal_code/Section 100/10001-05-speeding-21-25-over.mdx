# 100.01-05 - SPEEDING 21-25 OVER

Operating a motor vehicle above the posted speed limit by 21 to 25 mph.

<Aside type="caution" title="Charge Type: Misdemeanor">
- Classified as a **Misdemeanor** under this code entry.
- Driver license points on conviction: **3**.
- Base fine: **$350** (before fees/taxes if any).
</Aside>

## Elements
- Operates or controls a motor vehicle or engages in the described conduct.
- On a public roadway subject to a posted speed limit.
- Exceeds the posted speed limit by between 21 and 25 mph, inclusive.

## Penalties
- **Fine**: $350
- **License Points**: 3

## Example
Example: Driving 23 mph over the limit in a posted zone.
