## Why Use wLauncher?  
Our emergency vehicles need a **special siren tool** in the FiveM platform.  
**wLauncher** helps set up this tool quickly and easily.  

## How to Get wLauncher  
Download the latest version here: 

<LinkButton href="https://github.com/WhitigolSoftware/wLauncher/releases/latest/download/wLauncher.exe">Download wLauncher</LinkButton>

## Finding FiveM.exe  
When you open **wLauncher** for the first time, you need to **find** your `FiveM.exe` file.  

**Where is it usually located?** 
```plaintext
%localappdata%\FiveM
```

<Aside type="tip">
Copy and paste the path above into your File Explorer‚Äôs address bar to find it quickly.
</Aside>

### How to Import Your Profile:
<Steps>
1. Download the profile file from our **Discord server**.  
2. Open **wLauncher**.  
3. Click **"New Profile"**.  
4. Select **"Import Profile"**.  
5. Find the file you downloaded and select it. 
</Steps>

‚úÖ **Done!** Your profile is now set up, and you can use FRP.

## Important Notes

<Aside type="caution" title="Antivirus Warnings">
- Some antivirus programs **may say wLauncher is unsafe**, but this is a mistake.  
- If this happens, **allow the file** in your antivirus settings.
</Aside>

<Aside type="caution" title="Browser Warnings">

- If this happens, you will need to **manually allow it**.
