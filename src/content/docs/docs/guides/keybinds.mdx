## Keybinds

| **Keyboard Key**              |     | **Description**                |
| ----------------------------- | --- | ------------------------------ |
| <Kbd windows="F1" /> (hold)   |     | LEO Menu                       |
| <Kbd windows="F3" />          |     | EUP Menu for LEO               |
| <Kbd windows="F4" />          |     | Emote Menu                     |
| <Kbd windows="F7" />          |     | Open Phone                     |
| <Kbd windows="F8" />          |     | Open Client Console            |
| <Kbd windows="F9" />          |     | Vehicle Menu                   |
| <Kbd windows="F10" />         |     | Vehicle Damage Menu            |
| <Kbd windows="F11" />         |     | Change Proximity Chat Distance |
| <Kbd windows="M" />           |     | vMenu                          |
| <Kbd windows="N" />           |     | PTT (Voice Chat)               |
| <Kbd windows="T" />           |     | Text Chat                      |
| <Kbd windows="Z" />           |     | Show Money                     |
| <Kbd windows="CTRL+Z" />      |     | Put your seatbelt on           |

# Commands

| **Command**                           |     | **Description**                                                             |
| ------------------------------------- | --- | --------------------------------------------------------------------------- |
| <Kbd windows="/showcad" />            |     | Display CAD Interface                                                       |
| <Kbd windows="/repair" />             |     | Repair Vehicle                                                              |
| <Kbd windows="/heal" />               |     | Heal Player                                                                 |
| <Kbd windows="/asl" />                |     | ASL Functionality                                                           |
| <Kbd windows="/cadrefresh" />         |     | Refresh CAD Interface                                                       |
| <Kbd windows="/manageme" />           |     | Allows you to create/clone MP characters                                    |
| <Kbd windows="/mycar" />              |     | Set your car as yours, enabling locking functions                           |
| <Kbd windows="/radio" />              |     | Open the radio                                                              |
| <Kbd windows="/loadout" />            |     | Will give you a standard issue LEO loadout                                  |
| <Kbd windows="/unrack" />     |     | Will allow you to unrack a weapon from a police vehicle                     |
| <Kbd windows="/rack" />       |     | Will allow you to rack a weapon from a police vehicle                       |
| <Kbd windows="/timingtool" />          |     | Will give you a timing tablet for use with getting info for a vehicle's speed |
| <Kbd windows="/closetimingtool" />     |     | Will close the above tablet                                                 |
| <Kbd windows="/interiormanager" />     |     | Will open the Interior Manager to change IPLs/Entity Sets                   |
